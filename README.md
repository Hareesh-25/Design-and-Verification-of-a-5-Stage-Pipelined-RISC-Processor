# Design-and-Verification-of-a-5-Stage-Pipelined-RISC-Processor
Designed and verified a 5-stage pipelined RISC processor in Verilog HDL featuring IF, ID, EX, MEM, and WB stages. Implemented pipeline registers, hazard detection, and forwarding logic to improve instruction throughput. Functional verification was performed using simulation and testbench-based validation.
